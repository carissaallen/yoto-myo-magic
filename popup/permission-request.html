Additional Permission Required

The podcast you're trying to import has audio files hosted on external servers.

Why is this needed?
Some podcasts host their audio files on servers outside of Listen Notes. To download these files, the extension needs permission to access those external domains.

This permission will allow the extension to:

  • Download podcast audio files from any server
  • Follow redirects to the actual audio files
  • Import podcasts from various hosting providers

Note: This permission is only used when importing podcasts. The extension won't access any other websites or your browsing data.
